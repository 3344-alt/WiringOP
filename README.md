# wiringBP README

This is a modified WiringPi for OrangePi. We call it WiringOP.

## Download
### For Orangepi Pi
    git clone https://github.com/zhaolei/WiringOP.git -b h3 
## Installation
    cd WiringOP
    chmod +x ./build
    sudo ./build
    

And the source directory examples also has many demo.
